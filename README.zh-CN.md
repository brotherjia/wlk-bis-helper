# WLK 团本 BIS 助手 V1.0

这是一个给公会成员查询 WLK 团本 BIS 装备用的静态网页工具。打开网页后，选择阶段、人数、难度、职业和专精，就可以看到对应条件下的 BIS 装备列表。

## 主要功能

- 支持 P1、P2、P3、P4 阶段筛选
- 支持 10 人、25 人团本区分
- 支持普通、英雄难度区分
- 支持按职业和专精查询
- 按副本和 Boss 分组显示 BIS 装备
- 支持按 Boss、装备名、装备部位搜索
- 可以一键复制当前 BIS 清单，方便发到群里
- 纯网页运行，不需要安装软件，不需要登录

## 使用方法

打开网页后，在左侧依次选择：

1. 阶段，例如 P1、P2、P3、P4
2. 人数，例如 10 人或 25 人
3. 难度，例如普通或英雄
4. 职业
5. 专精

选择完成后，右侧会自动显示符合条件的 BIS 装备。结果会按照“副本 -> Boss”分组，方便打团时快速查看哪个 Boss 掉什么装备。

## 给公会成员使用

推荐把这个网页上传到 GitHub Pages、Gitee Pages、腾讯云 COS、阿里云 OSS、七牛云等静态网站服务。上传成功后，把公开链接发给公会成员即可。

如果只是临时使用，也可以把网页文件打包发给大家，让大家在电脑上直接打开 `index.html`。

## 需要上传的文件

如果上传到网站，至少需要上传下面几个文件：

- `index.html`
- `styles.css`
- `app.js`
- `data.js`

如果上传到 GitHub，也可以一起上传：

- `README.md`
- `README.zh-CN.md`

## 数据说明

装备数据基于 WLK Classic 的 Wowhead BIS 资料整理。为了保证装备名称准确，装备名保留英文；页面界面、副本、Boss、职业、专精、装备部位和属性标签使用中文显示。

这个工具适合打团时快速查询和分配装备参考，但如果涉及重要装备分配，建议再结合当前版本资料、公会规则和团员实际配装确认。
